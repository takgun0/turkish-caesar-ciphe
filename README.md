🛡️ Turkish Caesar Cipher Tool
A Python-based implementation of the classic Caesar Cipher algorithm, specifically optimized to support the Turkish alphabet and maintain case sensitivity.

📖 Description
This tool allows users to encrypt and decrypt text by shifting characters through the alphabet. Unlike standard implementations, this version correctly handles unique Turkish characters (ç, ğ, ı, ö, ş, ü) and preserves the original capitalization of the input text.

✨ Features
Full Turkish Support: Includes all 29 letters of the Turkish alphabet.

Case Sensitivity: "AbC" remains "AbC" after shifting (it doesn't force everything to lowercase).

Bidirectional: Supports both encryption and decryption.

Symbol Preservation: Numbers, spaces, and punctuation marks remain unchanged.

🚀 How to Run
Ensure you have Python 3.x installed.

Clone this repository:
git clone https://github.com/yourusername/turkish-caesar-cipher.git

Run the script:
python caesar_cipher.py



🛡️ Türkçe Sezar Şifreleme Aracı
Klasik Sezar Şifreleme algoritmasının, Türk alfabesine özel olarak optimize edilmiş ve büyük/küçük harf duyarlılığını koruyan Python uygulamasıdır.

📖 Açıklama
Bu araç, karakterleri alfabe üzerinde kaydırarak metinleri şifrelemenize veya şifreli metinleri çözmenize olanak tanır. Standart uygulamaların aksine, bu versiyon Türkçe'ye özgü karakterleri (ç, ğ, ı, ö, ş, ü) doğru şekilde işler ve metnin orijinal büyük/küçük harf yapısını bozmaz.

✨ Özellikler
Tam Türkçe Desteği: Türk alfabesindeki 29 harfin tamamını içerir.

Büyük/Küçük Harf Koruması: "AbC" gibi girişler, kaydırma sonrası biçimini korur.

Çift Yönlü İşlem: Hem şifreleme hem de şifre çözme desteği.

Sembol Koruma: Sayılar, boşluklar ve noktalama işaretleri işlemden etkilenmez.
